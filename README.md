# Glorious Destiny Advisory

Faith-driven financial education and mentorship—**wealth, leadership, legacy**—with **Sista Grace Ajayi** (WFG licensed professional).

## Stack

- [Astro](https://astro.build/) 5 + Tailwind 3  
- Static build (`output: "static"`)

## Setup

```bash
npm install
npm run dev
npm run build
```

## Config

Edit `src/config.json` for contact info and scheduler URL.
