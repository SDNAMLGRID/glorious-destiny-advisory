# Glorious Destiny Advisory

Faith-driven financial education and mentorship—**wealth, leadership, legacy**—with **Dr Gloria Ajayi** (WFG licensed professional).

## Stack

- [Astro](https://astro.build/) 5 + Tailwind 3  
- Static build (`output: "static"`)

## Setup

```bash
npm install
npm run dev
npm run build
```

## Config

Edit `src/config.json` for contact info. Set `calendlyUrl` to your embed URL (for example `https://calendly.com/your-handle/discovery`) to show the scheduler on `/book`; leave empty for the email fallback.
